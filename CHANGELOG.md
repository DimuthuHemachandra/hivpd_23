# Changelog

All notable changes to the HIVPD_23 Research Project website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-20

### Added
- Initial website launch
- Hero section with project overview and animated statistics
- Project goals section with 6 primary objectives
- Interactive patient recruitment statistics dashboard
  - Age distribution bar chart
  - Gender distribution doughnut chart
  - Ethnicity distribution pie chart
  - Recruitment timeline line graph
  - Site locations table
- Inclusion and exclusion criteria sections with expandable details
- Comprehensive data management protocols
  - Data collection procedures
  - Security measures
  - Privacy protection
  - Data access policies
  - Storage and backup information
  - Quality assurance processes
- MRI data protocols with collapsible accordion
  - Scanner specifications
  - T1-weighted imaging parameters
  - Diffusion weighted imaging (DWI) protocols
  - Functional MRI (BOLD) protocols
  - Quality control procedures
  - Image processing pipeline
- Metadata protocols and BIDS compliance documentation
- Funding and acknowledgments section
- References section with key publications
- Responsive navigation with mobile menu
- Dark/light mode toggle
- Smooth scrolling navigation
- Animated counters for statistics
- Back to top button
- Fully responsive design (mobile, tablet, desktop)
- WCAG 2.1 AA accessibility compliance
- Sample data files (demographics.json, recruitment.json)

### Technical Features
- HTML5 semantic markup
- CSS3 with CSS variables for theming
- Vanilla JavaScript (ES6+)
- Chart.js for data visualizations
- Font Awesome icons
- Google Fonts (Inter)
- GitHub Pages compatible
- BIDS-compliant data structure documentation

### Documentation
- README.md with setup and deployment instructions
- CONTRIBUTING.md with contribution guidelines
- LICENSE (MIT)
- CNAME.example for custom domain setup
- Comprehensive code comments
