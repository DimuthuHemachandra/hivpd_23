# Images Directory

Place your project images in this directory.

## Recommended Images

1. **favicon.png** - Website favicon (32x32 or 64x64 pixels)
2. **logo.png** - Project logo (for hero section)
3. **institution-logos/** - Logos for funding institutions
4. **team-photos/** - Research team member photos (if needed)

## Image Guidelines

- Use optimized images (compressed PNG, JPG, or WebP)
- Recommended maximum width: 2000px for hero images
- Use descriptive filenames (e.g., `harvard-logo.png` instead of `img1.png`)
- Include alt text when adding images to HTML

## Example Usage

```html
<img src="assets/images/logo.png" alt="HIVPD_23 Project Logo">
```
